Legal tags

make_legal_tags

This is the basic command that compiles a list of legal tags, divided into part-of-speech tags and syntactic tags. More precisely, the division is into preterminals and higher nodes. Part-of-speech tags can therefore appear on the list of syntactic tags if they occur recursively, as they do, say, in conjunction contexts. 
make_lexicon: t
Like any other command file, a legal tags query must end in .q and writes output to an .out file. But legal tags queries do not contain a boundary node.
